# Pavlov Photography

Статичен сайт за представяне на фотографски и видеографски услуги на Pavlov Photography.

## Файлове

- `index.html` — основната страница
- `style.css` — всички стилове и responsive правила
- `script.js` — интеракции, навигация, слайдер, FAQ и Formspree формата
- `vercel.json` — минимална конфигурация за Vercel

## Технологии

- Чист HTML
- Чист CSS
- Vanilla JavaScript
- Google Fonts
- Formspree за inquiry формата

## Форма за запитване

Формата е свързана към:

`https://formspree.io/f/mnjoooke`

Няма нужда от backend или build step.

## Локален преглед

Отворете `index.html` директно в браузър или пуснете папката през статичен локален сървър.

## Deploy във Vercel

1. Качете съдържанието на папката във Vercel проект.
2. Изберете `Other` / `No Framework`, ако бъдете попитани.
3. Не е необходим build command.
4. Не е необходим output directory.

## Бележки

- Сайтът е едностраничен и anchor навигацията работи без допълнителни библиотеки.
- Портфолио секцията е изградена с CSS placeholders и текст, които могат по-късно да се заменят с реални изображения и case studies.
- Имейлът и телефонът в контакт секцията са clickable.
